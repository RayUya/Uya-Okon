### 📌 Overview

**RNU-SPARK** is a next-generation, white-label university ecosystem mobile application. It is designed as a scalable, modular platform that universities can license and customize to provide a unified digital experience for their students. The app aims to consolidate academic, administrative, and social campus life into a single, intuitive interface.

### 🏛️ For Institutions

Universities can brand and configure RNU-SPARK to match their identity. The platform handles the heavy lifting while institutions get a custom app that their students will actually use.

### ✨ Core Modules

| Module | Description |
|--------|-------------|
| **Academic Hub** | Course materials, grades, attendance, and timetable access |
| **Administrative Portal** | Fee payments, registration, document requests |
| **Campus Life** | Event calendars, club listings, announcements |
| **Communication** | Push notifications, announcements, direct messaging |
| **Maps & Directory** | Campus navigation and staff/faculty directory |

### 🛠️ Tech Stack

- **Frontend:** Native Android (Java/XML)
- **Backend:** Firebase (Authentication, Firestore, Cloud Functions)
- **Notifications:** Firebase Cloud Messaging
- **Architecture:** Modular, scalable design for easy customization

### 🚀 Why RNU-SPARK?

Most universities either:
- ❌ Have no app at all
- ❌ Use fragmented solutions that don't talk to each other
- ❌ Pay massive amounts for custom development

**RNU-SPARK solves this** by offering a cost-effective, ready-to-deploy platform that feels custom-built.

### 📱 Features

- **White-label architecture** – Complete branding customization per institution
- **Scalable infrastructure** – Built to handle thousands of concurrent users
- **Modular design** – Institutions enable only what they need
- **Real-time updates** – Push notifications and live content

### ROBAN STORES 🛍️

Roban Stores is a modern shopping mall in Enugu offering a wide range of products including fashion, electronics, home goods, and more. Since they didn't have a mobile presence, I built this app to bring the Roban shopping experience directly to customers' pockets.

### 📱 About The Project

This is a custom-built mobile application designed to digitize the Roban Stores shopping experience. The app allows users to browse products, discover ongoing promotions, and stay connected with Enugu's favorite retail destination — all from their mobile device.

### ✨ Features

- **Product Catalog** – Browse items across categories like fashion, electronics, home & living, and more
- **In-Store Promotions** – Stay updated on the latest deals and discounts available at the Enugu mall
- **Store Information** – Location details, opening hours, and contact information for Roban Stores
- **Smooth UI** – Clean, intuitive interface designed with local shoppers in mind
